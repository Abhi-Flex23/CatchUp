FinnOS

FinnOS is an official product by the makers of CatchUp.

About:

FinnOS is a Mobile Operating System built and developed by Finn Tech.

FinnOS is based on OpenHarmony which is an Operating System made with OpenAtom.

FinnOS is yet in it's early stages of development and it will be full of new features and it will be optimised for lower end phones.

FinnOS has goals to make mobiles last longer and make everything smooth and optimised.

History:

The project started in January 2024, originally named OptiFlexOS. The Operating System wasn't based on anything, instead the OS would have had it's own base.

OptiFlexOS's UI was written in pure HTML and the first iteration wanted to ship the OS as a web-based OS.

On February till March, the name changed to OptiOS, and the OS still was web-based but this second iteration was based on Capyloon, another Mobile OS with the goal of providing Decentralised Web and a fast and secure experience.

On mid March stuff changed, and OptiFlex closed. Finn Tech was founded and so started FinnOS. FinnOS was still using code from it's roots - OptiOS.

On the month of May everything changed and the future of FinnOS was set, the new base is OpenHarmony and FinnOS is going native, as Finn Tech we're excited for the future!
